Pusat Bantuan

Selamat datang di pusat bantuan admin SMP NU Tugasari. Berikut panduan penggunaan sistem:

Bagaimana cara menambahkan siswa baru?
Buka menu Siswa di sidebar, lalu klik tombol Tambah Siswa di pojok kanan atas. Isi formulir (NIS, nama, kelas, jenis kelamin, wali, status). Anda juga dapat mengupload foto siswa langsung dari perangkat Anda. Klik Simpan untuk menyimpan data.
Bagaimana cara menyetujui pendaftar PPDB?
Buka menu PPDB, cari pendaftar dengan status Pending, lalu klik ikon centang hijau untuk menyetujui atau ikon X merah untuk menolak. Anda juga dapat mengedit detail pendaftar dengan mengklik ikon pena.
Bagaimana cara membalas pesan dari pengunjung?
Buka menu Pesan, pilih pesan dari daftar inbox di sebelah kiri, lalu klik tombol Balas untuk membuka aplikasi email Anda dengan pesan otomatis.
Bagaimana cara mengupload foto ke galeri?
Buka menu Galeri, klik Upload Foto. Pilih file gambar dari perangkat Anda, isi judul, pilih kategori (Akademik, Keagamaan, Ekstrakurikuler), dan tambahkan deskripsi. Klik Simpan.
Bagaimana cara mengubah pengaturan sekolah?
Buka menu Pengaturan. Anda dapat mengubah profil sekolah, pengaturan PPDB, media sosial, dan keamanan. Jangan lupa klik Simpan Perubahan setelah selesai.
Apakah data tersimpan otomatis?
Ya, semua perubahan disimpan otomatis di penyimpanan lokal browser (localStorage) dan akan disinkronkan ke database Supabase jika terhubung. Pastikan untuk tidak menghapus cache browser agar data tidak hilang.
Butuh Bantuan Lebih Lanjut?

Hubungi dukungan teknis di support@smpnutugasari.sch.id atau telepon (0281) 123456 ext. 102
